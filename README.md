# 🧾 电子发票自动拼版与粘贴单合并工具 (Invoice Auto Merger)

这是一个基于 Python 的轻量级办公自动化工具，专为解决财务报销中繁琐的发票打印问题而生。

它可以自动将**横向的电子发票**按“一页两张”的格式完美拼版到**竖向 A4 纸**上，并在每页拼好的发票后**自动插入一页指定的报销粘贴单**，让你告别手动排版的折磨，实现一键生成报销文件！

## ✨ 核心功能

* **🤖 智能缩放排版**：无视原发票内部复杂的旋转或坐标属性，利用 PyMuPDF 强制将两张发票等比缩小（70.7%）并上下对齐到竖向 A4 纸中。
* **📑 自动穿插粘贴单**：在生成的每一页发票拼版后，程序会自动抓取并追加一页《报销粘贴单》，排版顺序直接满足财务装订要求。
* **📁 规范的目录管理**：发票源文件与粘贴单模板分文件夹独立存放，输出文件不会污染原文件夹。

## 📂 项目结构说明

在使用本工具前，请确保你的文件夹结构如下：

```text
invoice-auto-merger/
  ├── fapiao_process.ipynb     # 核心运行代码 (Jupyter Notebook 格式)
  ├── 待报销的发票/            # 👈 请将你需要处理的 PDF 发票全部放在这里
  └── 粘贴单/
       └── 原始票据（_影像化）报销粘贴单.pdf  # 👈 你的空白粘贴单模板

## 🛠️ 环境依赖与安装

本项目依赖 `PyMuPDF` 处理 PDF 文件。请在你的命令行或终端中运行以下命令安装必备库：

```bash
pip install PyMuPDF
```

## 🚀 如何使用

1. **准备发票**：将所有需要报销的 PDF 电子发票放入 `待报销的发票` 文件夹中。
2. **准备模板**：确认 `粘贴单` 文件夹中存在名为 `原始票据（_影像化）报销粘贴单.pdf` 的文件（如需更改名字，请在代码中同步修改路径）。
3. **运行代码**：
   * 打开 `fapiao_process.ipynb`。
   * 运行全部代码块。
4. **获取结果**：运行完成后，项目主目录下会自动生成一个名为 **`最终_合并后的发票_带粘贴单.pdf`** 的文件，直接拿去打印机双面/单面打印即可！
